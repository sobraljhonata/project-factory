# CLI Architecture

## Objetivo
Permitir geração interativa de projetos a partir de templates e presets.

## Componentes
- entrada CLI
- prompts interativos
- resolvedor de opções
- composição de templates
- pós-processamento

## V1
- seleção de nome do projeto
- seleção de stack de API (fixa na V1)
- seleção de módulos de infra AWS
- geração de projeto em diretório final
